Add GET /api/india-risk-score to server.py. Cache 600s.
4 CONTRIBUTING FACTORS:
1. event_volume: GDELT India-relevant events last 24h → normalize 0-100
2. avg_severity: avg absolute Goldstein → severity=max(0,-avg)*10 → 0-100
3. market_stress: (VIX_curr - VIX_20d_avg)/VIX_20d_avg*100 → clamp 0-100
4. keyword_density: count danger words in last 20 RSS headlines
   words=['sanctions','war','conflict','crisis','strike','tension',
          'default','recession','crash','selloff','outflow','ban']
   score = min(100, count*8)
IRS = event_volume*0.30 + avg_severity*0.25 + market_stress*0.25 + keyword_density*0.20
Clamp 0-100. Round to 1 decimal.
MODE: >=65→RISK OFF | 40-64→NEUTRAL | <40→RISK ON
ZONE: >=80→EXTREME | 60-79→ELEVATED | 35-59→MODERATE | <35→LOW RISK
Include IRS in every WebSocket broadcast.

This system DETECTS anomalies. It does NOT predict price.
PREDICTION: 'Price will fall to X' — uncertain, often wrong
DETECTION: 'Unusual put buying happening right now' — observable fact

That was DETECTABLE. The ₹3800→₹1500 crash was not PREDICTABLE.
Your system fires when 3+ anomaly layers converge simultaneously.

async def compute_danger_score(symbol: str) → dict:
  # Fetch all 5 layers in parallel
  options, macro, legal, smart, sentiment = await asyncio.gather(
    get_options_anomaly(symbol),
    get_macro_pressure(symbol),
    get_legal_radar(),           # same for all Adani stocks
    get_smart_money(symbol),
    get_sentiment_velocity(symbol)
  )
  layer_scores = {
    'options_anomaly':    options['anomaly_score'],
    'macro_pressure':     macro['macro_pressure_score'],
    'legal_risk':         legal['legal_risk_score'],
    'smart_money':        smart['smart_money_score'],
    'sentiment_velocity': sentiment['sentiment_velocity_score'],
  }
  # Count active layers (score > 50 = layer is active)
  active_layers = [k for k,v in layer_scores.items() if v > 50]
  active_count = len(active_layers)
  # Weighted danger score
  weights = {
    'options_anomaly': 0.30,    # highest weight — most forward-looking
    'macro_pressure':  0.20,
    'legal_risk':      0.25,    # second highest — event-driven
    'smart_money':     0.15,
    'sentiment_velocity': 0.10
  }
  danger_score = sum(layer_scores[k]*weights[k] for k in weights)
  danger_score = round(min(100, danger_score), 1)
  # CONVERGENCE MULTIPLIER — key insight
  # When 3+ layers fire together: multiply score
  # Single layer firing = noise. Multiple layers = signal.
  if active_count >= 4: danger_score = min(100, danger_score * 1.4)
  elif active_count >= 3: danger_score = min(100, danger_score * 1.2)
  # FINAL SIGNAL
  if danger_score >= 75: final_signal = 'EXIT'
  elif danger_score >= 55: final_signal = 'REDUCE'
  elif danger_score >= 35: final_signal = 'WATCH'
  else: final_signal = 'CLEAR'
  return {
    'symbol': symbol,
    'danger_score': danger_score,
    'final_signal': final_signal,
    'active_layers': active_layers,
    'active_count': active_count,
    'layers': layer_scores,
    'convergence_note': f'{active_count}/5 layers active',
    'last_time_3plus_fired': _get_historical_convergence(symbol),
    'timestamp': datetime.now(IST).isoformat(),
    'disclaimer': 'Detection system only. Not financial advice. Always use stop losses.'
  }
